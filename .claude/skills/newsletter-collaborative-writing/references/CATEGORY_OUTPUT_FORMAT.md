# Category Section Output Template

After collecting user input for a category, append this structure to the output file:

```markdown
## [Titolo della categoria]

### I Takeaways per gli AI Engineers

- **Takeaway 1:** [Brief point from user]
- **Takeaway 2:** [Brief point from user]
- **Takeaway 3:** [Brief point from user]

- **Action Items:**
  - [action item 1 from user]
  - [action item 2 from user]

### Cosa succede questa settimana?

*[La mia analisi - user's text with proofreading only]*

### I link della settimana

*[Summarized list of links with brief descriptions (20-30 words each)]*
```

## Notes
- **Takeaways**: Exactly 3, provided by user, apply proofreading only
- **Action Items**: Exactly 2, provided by user, apply proofreading only
- **La mia analisi**: User's text, apply proofreading only
- **Link summaries**: Maximum 20-30 words per link, summarize from input
- All links from the category must be included and clickable
